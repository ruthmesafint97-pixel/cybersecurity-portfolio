## 📂 My Projects

### 1. [Secure Network Design](https://github.com/ruthmesafint97-pixel/secure-network-design)
A network lab built in Cisco Packet Tracer. I configured a network for a company with separate areas (VLANs) for staff and guests, handled IP addresses automatically (DHCP), and added secure logins using a RADIUS server.

### 2. [Custom Wordlist Generator](https://github.com/ruthmesafint97-pixel/custom-wordlist-generator)
A script I wrote in Python. It takes simple words (like a name or a year) and mixes them up with substitutions (like changing 'a' to '@') to create password combinations. It's used to test if passwords are easy to guess.
